**Overview**
Makivan Consultant is a modern, responsive business consulting website designed to showcase professional consulting services, 
build client trust, and generate leads. 
The website presents a clean, professional image with intuitive navigation and compelling calls-to-action.

**Features**
Responsive Design – Fully optimized for desktop, tablet, and mobile devices

Modern UI/UX – Clean, professional layout with smooth animations

Service Showcase – Clear presentation of consulting services offered

Contact Forms – Lead generation forms for client inquiries

About Section – Company story, mission, and values

Testimonials – Client success stories and reviews

SEO Optimized – Structured for better search engine visibility

Fast Loading – Optimized performance for better user experience


**Technologies Used**
HTML5 – Semantic markup structure

CSS3 – Custom styling with Flexbox & Grid

JavaScript – Interactive elements and animations

Font Awesome – Icon library for visual enhancement

Google Fonts – Professional typography
