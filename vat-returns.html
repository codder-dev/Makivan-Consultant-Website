Overview

What is VAT Returns Filing?

VAT (Value Added Tax) returns filing is a monthly obligation for businesses registered for VAT in Kenya. It involves calculating the VAT collected on sales, VAT paid on purchases, and submitting the difference to KRA by the 20th of every month.

Is VAT Returns Filing right for you?

FAQ

Common questions about VAT returns.

When is the VAT returns deadline?

VAT returns must be filed by the 20th of every month. If the 20th falls on a weekend or public holiday, the deadline is extended to the next working day.

What happens if I file late?

Late filing attracts a penalty of KES 10,000 for the first month and KES 5,000 for each subsequent month, plus interest on any unpaid tax.

Do I need to file if I made no sales?

Yes, if you are registered for VAT, you must file a nil return even if you made no sales during the period.

How do I know if I need to register for VAT?

If your annual turnover exceeds KES 5 million, you are required to register for VAT. We can help you determine your obligation.

Can you help with VAT refunds?

Yes, we help you claim VAT refunds by ensuring your returns are accurate and all supporting documents are properly prepared.
